# **App Name**: Presell Genius

## Core Features:

- AI Presell Content Generation: Utilize Gemini 1.5 Flash to intelligently craft high-converting headlines, engaging body copy, and compelling calls-to-action, functioning as a smart tool that incorporates key selling points from user-provided input.
- Customization Input Fields: Provide a comprehensive UI for users to input detailed sales page text or descriptions, select a custom hexadecimal color for primary action buttons using a color picker, and specify the target URL for links.
- Live Page Preview: Render a real-time, dynamic preview of the generated presell page directly within the application, allowing users to visualize their output as they make adjustments to content and styling.
- Clean HTML Output: Generate optimized, clean, and semantically structured HTML code for the presell page, ensuring compatibility and ease of integration across various platforms.
- HTML Download Functionality: Enable users to effortlessly download the complete generated HTML file for their presell page with a single click, ready for immediate deployment.

## Style Guidelines:

- Primary Color: A sophisticated, deep sapphire blue (#2952A3) that evokes trust and professionalism, suitable for prominent interactive elements and strong typographic emphasis.
- Background Color: An ultra-light, subtle cool-gray blue (#ECF0F7) providing a clean, spacious canvas that enhances readability without distraction, creating a professional yet calm environment.
- Accent Color: A vibrant, clear cyan (#28B6CE) chosen to create energetic visual contrasts and draw attention to critical calls-to-action or highlighted information.
- Font Recommendation: 'Inter' (sans-serif) for both headlines and body text. Its modern, highly readable design ensures clarity and professionalism, making it suitable for both engaging titles and extended content sections.
- Utilize a consistent set of clean, minimalist vector icons. Icons should visually communicate functionality clearly, supporting a high-efficiency user workflow and reinforcing a modern, trustworthy aesthetic.
- Employ a responsive, card-based layout structure with ample whitespace to ensure excellent readability and intuitive navigation across all devices. Inputs should be logically grouped with clear labels.
- Incorporate subtle and swift micro-interactions for feedback on user actions (e.g., button clicks, form submissions). Animations should be non-distracting and designed to enhance the sense of responsiveness.
